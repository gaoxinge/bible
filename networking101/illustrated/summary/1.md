## correspond

```
用户进程：网络层，传输层，应用层
内核：网络层，传输层
驱动：链路层
网卡：物理层
```

## flow

- 链路层有mtu
- 网络层的mtu在68到65536之间
- 在网络层进行分片，传给下层
- 在网络层进行重组，传给上层

## name

```
应用层：数据
传输层：段
网络层：数据报，数据包
链路层：帧
```

## protocol

```
应用层：ping, telnet & rlogin, ftp, smtp, x, traceroute, dns, ftp, bootp, snmp, nfs | rpc
运输层：tcp, udp
网络层：ip, icmp, igmp
链路层：arp, rarp, 以太网, ieee 802, slip, 压缩的slip, ppp
物理层
```
