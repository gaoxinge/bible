### 连接管理

- 三次握手

```
       SYN     seq=x
client -----------------------> server
       SYN ACK seq=y   ack=x+1
client <----------------------- server
       ACK     seq=x+1 ack=y+1
client -----------------------> server 
```

- 四次挥手

```
       FIN ACK seq=
client ----------------------->
```
