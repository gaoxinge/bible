## 网卡

## windows

### ipconfig

ipconfig命令用于显示网卡的配置，可以自动刷新DHCP和DNS设置。例如

```
> ipconfig
```

仅显示有配置的网卡，而

```
> ipconfig /all
```

显示所有网卡的配置

### ping

ping命令通过发送ICMP回声请求报文来检测与另一台计算机的连接。例如

```
> ping 127.0.0.1
```
