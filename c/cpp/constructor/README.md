# 构造器

## 定义

|                 | 构造器   | 是否隐式 (explicit) | 参数类型是否可以转化 | 流程            |
|-----------------|----------|---------------------|----------------------|-----------------|
| `A a`           | 默认构造 | 否                  | 否                   | 构造            |
| `A a(1)`        | 转化构造 | 否                  | 是                   | 构造            |
| `A a(1, 2)`     |          | 否                  | 是                   | 构造            |
| `A a{1}`        | 转化构造 | 否                  | 否                   | 构造            |
| `A a{1, 2}`     |          | 否                  | 否                   | 构造            |
| `A a = A(1)`    | 转化构造 | 否                  | 否                   | 构造, 构造/赋值 |
| `A a = A(1, 2)` |          | 否                  | 否                   | 构造, 构造/赋值 |
| `A a = {1}`     | 转化构造 | 是                  | 否                   | 构造, 构造/赋值 |
| `A a = {1, 2}`  |          | 是                  | 否                   | 构造, 构造/赋值 |
| `A a = 1`       | 转化构造 | 是                  | 否                   | 构造, 构造/赋值 |
